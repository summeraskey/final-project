# final-project

For our project, we want to analyze factors associated with trending Youtube videos using the Kaggle trending videos dataset. To expand our analysis, we will also collect additional information about the channels and videos using the Youtube Data API. After merging the datasets, we will analyze how channel features such as views, likes, comments, etc. relate to video engagement. The goal is to determine which features have the strongest relationships with video traffic in order to predict virality of youtube videos. We will develop a python package for collecting channel data, cleaning the merged dataset, and performing statistical EDA and visualization that shows trends in video popularity. 

Kaggle dataset: https://www.kaggle.com/datasets/rsrishav/youtube-trending-video-dataset/data
Youtube API: https://developers.google.com/youtube/v3 

Possible research questions: 
Do channels with more subscribers appear more in trending videos? 
Does channel country affect trending success or engagement? 
What video duration produces the most engagement (views, likes, comments)? 
Which category of video is most likely to appear in the top trending videos?
Does posting earlier vs. later in the week affect traffic?

Workflow: Kaggle dataset extraction → collect channel data through API → clean and merge → analyze and visualize.
